# Revisão de internações hospitalares por pneumonia com Machine Learning 

Projeto voltado à análise preditiva de dados públicos de saúde, com foco na compreensão e previsão das taxas 
de internações hospitalares por pneumonia em municípios brasileiros:

• Análise e tratamento de dados utilizando Python (Pandas, NumPy). 

•  Execução do projeto em ambiente Google Colab, com versionamento e armazenamento dos dados no 
Google Drive. 

• Integração de múltiplas bases públicas (DATASUS e IBGE), envolvendo dados demográficos, 
socioeconômicos, climáticos e territoriais. 

• Desenvolvimento de modelo preditivo com Random Forest Classifier, utilizando classificação por quartis. 

• Avaliação do modelo, atingindo acurácia média de 0,76, com bom desempenho em classes majoritárias. 

• Geração de visualizações analíticas com Matplotlib e Seaborn para apoio à interpretação dos resultados.
